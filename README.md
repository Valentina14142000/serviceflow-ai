# ServiceFlow AI

ServiceFlow AI is an intelligent triage and dispatch engine designed for automating service requests through AI-powered analysis.

## Features
- **Intelligent Triage**: Uses LLM-driven structured output to categorize incoming requests.
- **API-First Design**: Built with FastAPI for high-performance integration.
- **Extensible Architecture**: Designed for modular integration of agents and automation workflows.

## Technical Stack
- **Framework**: FastAPI
- **AI/LLM**: LangChain, OpenAI
- **Language**: Python 3.12+
